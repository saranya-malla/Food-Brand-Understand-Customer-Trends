# 🍔 Food Trends – Understanding Customer Preferences

## 📌 Project Overview

This project focuses on analyzing customer food preferences and understanding trends using both **structured survey data** and **unstructured social media data (Facebook posts)**.
The goal is to extract meaningful insights about customer behavior, preferences, and brand perception.

---

## 🎯 Objectives

* Analyze customer food preferences and choices
* Identify popular food categories and products
* Understand customer satisfaction (taste, price, quality)
* Perform sentiment analysis using social media data
* Compare survey insights with real-world engagement data

---

## 📊 Data Sources

### 1. Structured Data (Survey)

* Collected using **Google Forms**
* Includes:

  * Age, Gender, City
  * Food Preferences (Veg/Non-Veg)
  * Favorite Food Items
  * Ratings (Taste, Price, Quality)
  * Purchase Behavior
  * Customer Feedback

### 2. Unstructured Data (Facebook)

* Scraped from public food brand pages
* Includes:

  * Post content
  * Likes, Comments, Shares
  * Engagement metrics

---

## 🛠️ Tools & Technologies Used

* Microsoft Excel (Data Cleaning & Preparation)
* Power BI (Dashboard & Visualization)
* GitHub (Version Control)
* Google Forms (Data Collection)
* Apify (Facebook Data Scraping)

---

## 📈 Key Analysis Performed

* Customer Segmentation (Age, Gender, Preferences)
* Product-Level Analysis
* Rating Analysis (Taste, Price, Quality)
* Sentiment vs Engagement Analysis
* Brand Performance Comparison

---

## 📊 Dashboard Features

* KPI Cards (Total Customers, Avg Ratings)
* Interactive Filters (Age, Gender, Food Type)
* Charts & Graphs for trend analysis
* Comparison of Survey vs Social Media Insights

---

## 📂 Project Structure

* Structured_dataset.xlsx → Cleaned survey data
* Unstructured_Dataset.xlsx → Facebook data
* FoodTrends.pbix → Power BI Dashboard
* FoodTrends_PPT.pptx → Project Presentation

---

## 🔍 Key Insights

* Customer preferences vary significantly by age group
* Taste is the most influential factor in purchase decisions
* Certain brands show higher engagement but lower satisfaction
* Social media trends align partially with survey data

---

## 🚀 Conclusion

This project demonstrates how combining structured and unstructured data can provide deeper insights into customer behavior.
It helps businesses make data-driven decisions to improve products and marketing strategies.
